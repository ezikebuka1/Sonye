# D8.1 — Avatar Color Palette Extension

Status: Locked 2026-05-16
Amends: D8 (design system — "avatars: blue palette only" rule)
Triggered by: D7.3 (gender-derived avatar color)

## Decision

Avatar color encodes gender:
- Woman → pink family
- Man → blue family (existing D8 blue palette, unchanged)
- Non-binary / Prefer not to say / NULL → green
  (reuse existing D8 skill-badge green: bg #E8F5E9 / ink #27500A family)

Coral remains forbidden for avatars (coral is reserved for actions/CTAs
only — that D8 rule is unchanged).

## Pink palette

Specific pink hex values are TBD (mirror blue HSL — match the blue
palette's saturation/lightness so men's and women's avatars feel
visually equivalent, not muted-blue-vs-loud-pink). Finalized during M5
lobby polish. Until then, placeholder pink is acceptable; the schema
does not depend on exact shade, only "woman → pink family".

## Does NOT change

Coral action-only rule. Existing blue avatar shades. No new color
families invented beyond pink.

## Downstream

- M5 lobby polish: finalize pink hex values
- Avatar rendering: gender → color mapping (executed in pre-M3 Commit 2)
